# API Automation

## Tại sao cần API Automation?

- Regression testing nhanh chóng
- Tích hợp vào CI/CD pipeline
- Consistent và repeatable
- Coverage report tự động

## Tools phổ biến

| Tool | Language | Đặc điểm |
|---|---|---|
| **Rest Assured** | Java | Phổ biến nhất cho Java |
| **Supertest** | JavaScript | Tích hợp tốt với Node.js |
| **Requests + pytest** | Python | Đơn giản, mạnh mẽ |
| **Playwright API** | Multi-lang | Built-in API testing |

::: info
Nội dung chi tiết sẽ được bổ sung sau.
:::
