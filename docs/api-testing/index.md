# API Testing

API Testing là kiểm thử tầng business logic mà không cần qua giao diện, giúp phát hiện lỗi sớm và nhanh hơn UI testing.

## Nội dung

- [REST API Testing](./rest-api) - Kiến thức nền tảng về REST API testing
- [Postman](./postman) - Tool phổ biến nhất cho API testing
- [API Automation](./api-automation) - Tự động hóa API testing

## Tại sao API Testing quan trọng?

1. **Nhanh hơn** UI testing nhiều lần
2. **Ổn định hơn** - không bị ảnh hưởng bởi UI changes
3. **Phát hiện lỗi sớm** - test trước khi UI hoàn thành
4. **Coverage cao hơn** - test được edge cases khó test qua UI

Nội dung chi tiết sẽ được bổ sung sau.
