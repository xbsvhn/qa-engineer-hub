# Career Path

## Lộ trình phát triển QA Engineer

```
Junior QA → QA Engineer → Senior QA → QA Lead → QA Manager
                                    → SDET
                                    → QA Architect
```

## Skills theo Level

### Junior QA (0-2 năm)
- Manual testing fundamentals
- Bug reporting
- Test case design
- Basic SQL
- Basic API testing (Postman)

### QA Engineer (2-4 năm)
- Test automation (1-2 tools)
- API testing proficient
- CI/CD basics
- Performance testing basics
- Mentoring juniors

### Senior QA (4-7 năm)
- Automation framework design
- Test strategy
- Performance & Security testing
- Technical leadership
- Process improvement

### QA Lead / Manager (7+ năm)
- Team management
- Resource planning
- Quality metrics & KPIs
- Cross-team collaboration
- Hiring & training

::: info
Nội dung chi tiết sẽ được bổ sung sau.
:::
