# QA Mindset

## Tư duy của QA chuyên nghiệp

### 1. Think like a user
- Đặt mình vào vị trí người dùng
- Hỏi "Nếu tôi là user, tôi sẽ dùng feature này như thế nào?"

### 2. Think like a hacker
- Tìm cách phá hệ thống
- "Nếu tôi muốn làm hỏng feature này, tôi sẽ làm gì?"

### 3. Question everything
- Không assume requirement đúng
- "Requirement nói vậy, nhưng user thực sự cần gì?"

### 4. Attention to detail
- Chú ý đến từng chi tiết nhỏ
- UI alignment, typo, edge cases

### 5. Big picture thinking
- Hiểu feature trong context toàn bộ hệ thống
- Impact analysis khi có thay đổi

::: info
Nội dung chi tiết sẽ được bổ sung sau.
:::
