# Fundamentals

Phần này bao gồm các kiến thức nền tảng mà mọi QA Engineer cần nắm vững.

## Nội dung

- [Testing Concepts](./testing-concepts) - Các khái niệm cơ bản về kiểm thử phần mềm
- [SDLC & STLC](./sdlc-stlc) - Vòng đời phát triển và kiểm thử phần mềm
- [Test Design Techniques](./test-design-techniques) - Kỹ thuật thiết kế test case
- [Testing Levels](./testing-levels) - Các cấp độ kiểm thử
- [Testing Types](./testing-types) - Các loại kiểm thử

## Tại sao cần nắm vững Fundamentals?

Fundamentals là nền tảng giúp bạn:

1. **Hiểu bản chất** của việc kiểm thử phần mềm
2. **Giao tiếp chuyên nghiệp** với team bằng thuật ngữ chuẩn
3. **Áp dụng đúng kỹ thuật** cho từng tình huống cụ thể
4. **Phát triển career** lên các vị trí Senior, Lead, Manager

::: tip Lời khuyên
Dù bạn làm Manual hay Automation, Fundamentals luôn là phần quan trọng nhất. Hãy dành thời gian học kỹ trước khi đi sâu vào các chuyên đề khác.
:::
