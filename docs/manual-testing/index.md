# Manual Testing

Kỹ năng Manual Testing là nền tảng của mọi QA Engineer, dù bạn có chuyển sang Automation thì Manual vẫn luôn cần thiết.

## Nội dung

- [Test Plan & Strategy](./test-plan) - Lập kế hoạch và chiến lược kiểm thử
- [Test Case Design](./test-case-design) - Viết test case chuyên nghiệp
- [Bug Reporting](./bug-reporting) - Báo cáo bug hiệu quả
- [Exploratory Testing](./exploratory-testing) - Kiểm thử khám phá

::: tip
Manual Testing không phải là "click bừa". Đó là một quy trình có hệ thống, đòi hỏi tư duy phân tích và kỹ năng thiết kế test case.
:::
