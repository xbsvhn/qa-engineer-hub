# Performance Concepts

## Metrics quan trọng

| Metric | Mô tả | Target thường gặp |
|---|---|---|
| **Response Time** | Thời gian phản hồi | < 2 giây |
| **Throughput** | Số requests/giây | Phụ thuộc hệ thống |
| **Error Rate** | Tỷ lệ lỗi | < 1% |
| **Concurrent Users** | Số user đồng thời | Theo requirement |

## Performance Testing Process

1. **Identify test environment** - Env giống production nhất có thể
2. **Define acceptance criteria** - Response time, throughput targets
3. **Plan & design tests** - Scenarios, test data, workload model
4. **Configure test environment** - Setup tools, monitoring
5. **Execute tests** - Run và collect results
6. **Analyze & report** - Bottlenecks, recommendations

::: info
Nội dung chi tiết sẽ được bổ sung sau.
:::
