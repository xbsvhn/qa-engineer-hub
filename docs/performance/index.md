# Performance Testing

Performance Testing đánh giá tốc độ, khả năng mở rộng và độ ổn định của hệ thống dưới tải.

## Nội dung

- [Performance Concepts](./concepts) - Khái niệm cơ bản
- [JMeter](./jmeter) - Apache JMeter
- [K6](./k6) - Modern load testing tool

## Các loại Performance Testing

| Loại | Mục đích |
|---|---|
| **Load Testing** | Test với tải dự kiến |
| **Stress Testing** | Test vượt quá giới hạn |
| **Spike Testing** | Test với tải tăng đột ngột |
| **Endurance Testing** | Test trong thời gian dài |
| **Scalability Testing** | Đánh giá khả năng mở rộng |

::: info
Nội dung chi tiết sẽ được bổ sung sau.
:::
