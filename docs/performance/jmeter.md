# JMeter

## Apache JMeter là gì?

JMeter là open-source tool phổ biến nhất cho performance testing, hỗ trợ nhiều protocols: HTTP, JDBC, SOAP, REST, FTP.

## Các thành phần chính

- **Test Plan** - Root element
- **Thread Group** - Mô phỏng users
- **Samplers** - HTTP Request, JDBC Request...
- **Listeners** - View Results, Summary Report
- **Assertions** - Validate response
- **Config Elements** - CSV Data, HTTP Header Manager

::: info
Nội dung chi tiết sẽ được bổ sung sau.
:::
