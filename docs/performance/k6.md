# K6

## Ban chat: K6 = JMeter phien ban "developer-friendly"

Neu JMeter la **may anh co dien** (GUI, nhieu nut, ai cung dung duoc), thi K6 la **may anh mirrorless** (nho gon, manh me, can biet cach dung).

> **K6** = tool load testing viet bang **JavaScript**. Nhe, nhanh, va tich hop CI/CD **cuc ky de**. Duoc Grafana Labs phat trien.

### Tai sao K6?

| Tieu chi | K6 | JMeter |
|---|---|---|
| **Scripting** | JavaScript (quen thuoc voi QA automation) | GUI-based (XML) |
| **CI/CD** | Native support -- chi can 1 lenh | Can plugin |
| **Performance** | Rat nhe, it ton RAM | Nang (Java-based) |
| **Code review** | Scripts trong Git, doc duoc | JMX files khong doc duoc |
| **Learning curve** | De neu biet JS | De neu thich GUI |

::: tip Aha moment
K6 scripts la JavaScript binh thuong -- ban da hoc JS cho Playwright, gio dung CUNG ngon ngu do de viet performance tests. Khong can hoc tool moi!
:::

## Test dau tien -- Giai thich tung dong

```javascript
// load-test.js
import http from 'k6/http';
// ^^^ Import module HTTP cua k6 -- dung de gui requests
import { check, sleep } from 'k6';
// ^^^ check = kiem tra response co dung khong
//     sleep = doi (mo phong nguoi dung that nghi giua cac thao tac)

// === CAU HINH: bao nhieu nguoi, chay bao lau ===
export const options = {
  vus: 10,            // 10 virtual users (10 "nguoi dung ao")
  duration: '30s',    // chay trong 30 giay
};

// === HAM TEST: moi virtual user chay ham nay LAP LAI ===
export default function () {
  // Gui GET request den server
  const res = http.get('https://api.example.com/products');

  // Kiem tra response
  check(res, {
    'status la 200': (r) => r.status === 200,
    // ^^^ Response status phai la 200 (OK)

    'response nhanh hon 500ms': (r) => r.timings.duration < 500,
    // ^^^ Thoi gian phan hoi phai < 500ms

    'co data tra ve': (r) => r.json().length > 0,
    // ^^^ Body phai co data (khong rong)
  });

  sleep(1);
  // ^^^ Doi 1 giay truoc khi lap lai
  // (mo phong nguoi dung that: doc trang roi moi click tiep)
}
```

```bash
# Chay test
k6 run load-test.js

# K6 se hien thi ket qua:
#   http_req_duration...: avg=234ms  p(90)=450ms  p(95)=680ms
#   http_req_failed.....: 0.00%
#   vus.................: 10
#   iterations..........: 280
```

## Load Patterns -- Mo phong cac kieu tai khac nhau

### Constant Load = "Giu nguyen so khach"

```javascript
// Don gian: 50 users chay 5 phut
export const options = {
  vus: 50,           // 50 nguoi dung ao
  duration: '5m',    // chay 5 phut
};
```

### Ramp-up / Ramp-down = "Tang dan, giam dan" (thuc te nhat)

```javascript
export const options = {
  stages: [
    // stages = cac "giai doan" -- giong nhac co nhieu doan
    { duration: '2m', target: 50 },   // 2 phut dau: tang dan tu 0 len 50 users
    { duration: '5m', target: 50 },   // 5 phut tiep: giu 50 users (do tai on dinh)
    { duration: '2m', target: 100 },  // 2 phut: tang len 100 users
    { duration: '5m', target: 100 },  // 5 phut: giu 100 users
    { duration: '2m', target: 0 },    // 2 phut cuoi: giam dan ve 0 (ket thuc)
  ],
};
```

### Spike Test = "Dot ngot tang vot"

```javascript
export const options = {
  stages: [
    { duration: '1m', target: 50 },    // Tai binh thuong
    { duration: '10s', target: 500 },   // SPIKE! tang tu 50 len 500 trong 10 giay
    { duration: '2m', target: 500 },    // Giu tai cao 2 phut
    { duration: '10s', target: 50 },    // Ha ve binh thuong
    { duration: '2m', target: 50 },     // Giu tai binh thuong -- he thong co recover?
  ],
};
```

## Thresholds = "Dieu kien PASS/FAIL" tu dong

Thresholds giong nhu ban noi truoc: "Neu P95 > 2 giay hoac error > 1% thi = FAIL". K6 se **tu dong** danh gia PASS/FAIL.

```javascript
export const options = {
  stages: [
    { duration: '2m', target: 100 },
    { duration: '5m', target: 100 },
    { duration: '1m', target: 0 },
  ],

  thresholds: {
    // === DIEU KIEN PASS/FAIL ===

    http_req_duration: [
      'p(90) < 1000',   // 90% requests phai nhanh hon 1 giay
      'p(95) < 2000',   // 95% requests phai nhanh hon 2 giay
    ],

    http_req_failed: [
      'rate < 0.01',    // Error rate phai < 1%
    ],

    // Threshold rieng cho tung endpoint
    'http_req_duration{name:login}': [
      'p(95) < 1500',   // Login API phai nhanh hon 1.5 giay
    ],
  },
};
```

### Ket qua khi PASS:

```
v  http_req_duration........: avg=450ms  p(90)=890ms  p(95)=1200ms
     v { name:login }.......: avg=380ms  p(95)=980ms
v  http_req_failed...........: 0.5%  v < 1%
```

### Ket qua khi FAIL:

```
x  http_req_duration........: avg=3200ms  p(90)=5000ms  p(95)=8000ms
     x p(95) < 2000
# K6 exit code = 99 (FAIL) --> CI/CD pipeline se FAIL
# --> Deploy bi BLOCK tu dong!
```

::: tip Aha moment
Thresholds + CI/CD = **tu dong chan deploy** khi performance khong dat. Khong can QA ngoi xem ket qua roi quyet dinh -- K6 quyet dinh giup ban.
:::

## Vi du thuc te: E-commerce Load Test

```javascript
// ecommerce-load-test.js
import http from 'k6/http';
import { check, group, sleep } from 'k6';
// ^^^ group = nhom cac requests lai thanh 1 scenario (de doc report)

const BASE_URL = 'https://api.example.com';

// === CAU HINH ===
export const options = {
  stages: [
    { duration: '2m', target: 50 },   // Tang dan len 50 users
    { duration: '5m', target: 50 },   // Giu 50 users trong 5 phut
    { duration: '1m', target: 0 },    // Giam ve 0
  ],
  thresholds: {
    http_req_duration: ['p(95) < 2000'],  // 95% requests < 2 giay
    http_req_failed: ['rate < 0.01'],      // Error < 1%
  },
};

// === SCENARIO: Nguoi dung xem san pham va them vao gio hang ===
export default function () {

  // Buoc 1: Xem danh sach san pham
  group('Xem danh sach san pham', () => {
    const res = http.get(`${BASE_URL}/api/products?page=1&limit=20`);
    check(res, {
      'danh sach tra ve 200': (r) => r.status === 200,
      'co san pham': (r) => r.json().data.length > 0,
    });
    sleep(2);
    // ^^^ Nguoi dung doc danh sach 2 giay truoc khi click vao san pham
  });

  // Buoc 2: Xem chi tiet 1 san pham
  group('Xem chi tiet san pham', () => {
    const res = http.get(`${BASE_URL}/api/products/1`);
    check(res, {
      'chi tiet tra ve 200': (r) => r.status === 200,
      'co ten san pham': (r) => r.json().name !== '',
    });
    sleep(1);
    // ^^^ Doc chi tiet 1 giay
  });

  // Buoc 3: Them vao gio hang
  group('Them vao gio hang', () => {
    const payload = JSON.stringify({
      productId: 1,
      quantity: 1,
    });
    const headers = { 'Content-Type': 'application/json' };

    const res = http.post(`${BASE_URL}/api/cart`, payload, { headers });
    check(res, {
      'them gio hang thanh cong': (r) => r.status === 200,
    });
    sleep(1);
  });
}
```

## CI/CD Integration -- Chay K6 tu dong trong GitHub Actions

```yaml
# .github/workflows/performance.yml
name: Performance Tests

on:
  schedule:
    - cron: '0 2 * * 1'
    # ^^^ Moi thu 2, 2AM UTC (9AM Vietnam)
    # Chay hang tuan de theo doi performance theo thoi gian

jobs:
  k6-test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      # Dung K6 action co san -- khong can cai dat gi
      - uses: grafana/k6-action@v0.3.1
        with:
          filename: tests/performance/load-test.js
          # ^^^ Duong dan den file test K6
        env:
          BASE_URL: https://staging.example.com
          # ^^^ Chay tren staging, KHONG chay tren production!

      # Luu ket qua de xem sau
      - uses: actions/upload-artifact@v4
        if: always()
        with:
          name: k6-results
          path: summary.json
```

::: tip Aha moment
K6 + GitHub Actions = **performance testing tu dong hang tuan**. Neu tuan nay P95 tang tu 800ms len 2000ms, ban biet NGAY -- thay vi doi user phan nan.
:::

## Tom tat chuong

| Concept | An du | K6 cach dung |
|---|---|---|
| **VUs** | Nguoi dung ao | `options.vus` hoac `stages` |
| **Duration** | Chay bao lau | `options.duration` |
| **Stages** | Cac giai doan (tang, giu, giam) | Ramp-up, sustain, ramp-down |
| **Checks** | Kiem tra response | `check(res, { ... })` |
| **Thresholds** | Dieu kien PASS/FAIL | `options.thresholds` -- tu dong fail CI/CD |
| **Groups** | Nhom requests | `group('ten', () => { ... })` |
| **Sleep** | Nguoi dung nghi | `sleep(1)` -- mo phong realistic |
| **CI/CD** | Chay tu dong | GitHub Actions + grafana/k6-action |
