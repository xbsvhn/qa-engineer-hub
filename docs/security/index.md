# Security Testing

Security Testing giúp phát hiện lỗ hổng bảo mật trước khi kẻ tấn công khai thác chúng.

## Nội dung

- [OWASP Top 10](./owasp-top-10) - 10 lỗ hổng bảo mật phổ biến nhất
- [Security Testing Tools](./tools) - Công cụ kiểm thử bảo mật

## Tại sao QA cần biết Security Testing?

- Bảo mật là **trách nhiệm của cả team**, không chỉ security team
- Nhiều lỗ hổng có thể phát hiện qua **basic security checks**
- Giúp bạn **nâng cao giá trị** và khác biệt với QA khác
