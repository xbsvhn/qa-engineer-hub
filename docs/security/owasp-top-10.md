# OWASP Top 10

## OWASP Top 10 (2021)

| # | Vulnerability | Mô tả |
|---|---|---|
| A01 | **Broken Access Control** | User truy cập dữ liệu/chức năng không được phép |
| A02 | **Cryptographic Failures** | Dữ liệu nhạy cảm không được mã hóa đúng cách |
| A03 | **Injection** | SQL, NoSQL, OS command injection |
| A04 | **Insecure Design** | Thiết kế thiếu security controls |
| A05 | **Security Misconfiguration** | Cấu hình mặc định, thiếu hardening |
| A06 | **Vulnerable Components** | Sử dụng thư viện có lỗ hổng |
| A07 | **Authentication Failures** | Xác thực yếu, brute force |
| A08 | **Data Integrity Failures** | Không verify integrity của data/code |
| A09 | **Logging & Monitoring Failures** | Thiếu log và giám sát |
| A10 | **SSRF** | Server-Side Request Forgery |

::: info
Nội dung chi tiết cho từng vulnerability sẽ được bổ sung sau.
:::
