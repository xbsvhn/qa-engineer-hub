# Security Testing Tools

## Tools phổ biến cho QA

| Tool | Loại | Mô tả |
|---|---|---|
| **OWASP ZAP** | DAST | Scan web app tìm lỗ hổng, miễn phí |
| **Burp Suite** | DAST | Proxy intercept và test security |
| **SonarQube** | SAST | Scan source code |
| **Snyk** | SCA | Check vulnerable dependencies |
| **Postman** | API | Test API security manually |

## SAST vs DAST

| | SAST (Static) | DAST (Dynamic) |
|---|---|---|
| **Khi nào** | Trong quá trình code | Khi app đang chạy |
| **Input** | Source code | Running application |
| **Ưu điểm** | Phát hiện sớm | Tìm runtime issues |

::: info
Nội dung chi tiết sẽ được bổ sung sau.
:::
